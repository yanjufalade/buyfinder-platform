# Core Domain (the reason the business exists)

For BuyFinder:

Purchase Intent
Supplier Matching
Offer Evaluation
Award Decision

If these disappeared, BuyFinder would cease to be BuyFinder.

# Supporting Domains

These are essential, but they support the core.

Identity
Payments
Notifications
Analytics
Administration

Without them, the platform becomes harder to use.

But it is still fundamentally BuyFinder.

# Generic Domains

Things we don't want to reinvent.

Email delivery
SMS
File storage
Payment gateway integration
Authentication libraries

We'll use existing solutions where appropriate.
